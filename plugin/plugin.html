Frequently Asked Questions

Answers to common queries about SaasAble.

How do I install the plugin?

While SaasAble is designed with SaaS applications in mind, its not limited to SaaS products. SaasAble is multipurpose and can be used for a variety of websites and applications, whether its for an agency, digital product, CRM, blog, AI platform, or any other system. SaasAble is versatile and adaptable to fit any use case, SaaS or not. Its built for all types of projects!

Is the plugin compatible with my website's theme?

SaasAble offers industry-specific templates, a React Material UI Kit, Figma UI Kit, and flexible customization options that cater to both front-end marketing and admin interfaces.

Can I customize the plugin's appearance?

Yes, SaasAble is built for both, with a Figma UI kit for designers and React code for developers.

  • Figma UI Kit for Designers.

  • React Material UI Code for Developers.

  • Seamless Collaboration.

How often are updates released for the plugin?

The Figma version is for design and prototyping, while the codebase version provides ready-to-use React components for development.

Can I upgrade my plan later?

You will receive the template in a .zip format immediately after successfully purchasing the template.

What's included in the SaasAble license?

It includes the Figma UI kit, React Material UI codebase, regular updates, and customer support.

Can I use SaasAble for multiple projects?

No, one license allows usage in a single end product. For each new product or project, you are required to purchase a separate license. This ensures compliance with the licensing terms and provides ongoing support for each individual product. Make sure to obtain the appropriate license for every new project you work on.

  • One License = One End Product

  • Separate License for Each Product

  • Comply with License Terms

Do you offer refunds?

Yes, we offer refunds, but only if you provide a valid reason to support your claim. For more details, please review our MUI Store Refund Policy.

Do I get access to future updates?

Yes, you will receive future updates for 1 year from the date of purchase. If you need updates after 1 year, you will need to purchase the license again. However, this is not a renewalyour license is valid for a lifetime, but updates are limited to the 1-year period.

  • 1 Year Free Updates

  • License validity is lifetime

Is support included with the template purchase?

Support is included for 6 months, but it is limited to resolving issues within the template itself. We do not provide support for installation, customization queries, or troubleshooting unrelated issues. For ongoing support beyond 6 months, you will need to extend the support period with an additional purchase.
